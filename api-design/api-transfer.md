# データ転送 API (v1)

## ファイルダウンロード
指定されたパスにあるファイルをダウンロードします。

### リクエスト
- **URL:** `/v1/file`
- **Method:** `GET`

#### クエリパラメータ

| パラメータ名 | 必須 | 型 | フォーマット | デフォルト値 | 値の例 | 説明 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| `path` | 〇 | `string` | 1文字以上の文字列 | - | `"data/sapmle"` | `DATA_ROOT` からの相対パス<br>以下の場合にはエラーとなる<br>・空文字<br>・絶対パス (`/`で始まる)<br>・`../`を含む |


### レスポンス
- **200 OK:** ファイルバイナリを返却。
- **Error:** [共通エラー形式](./api-endpoint.md) に基づく JSON を返却。
  - `type` 値は以下参照

#### このエンドポイント固有のエラー識別子 (type)
`type` URLの末尾には以下の識別子が設定されます。

| 識別子 (type末尾) | ステータス | 理由 |
| :--- | :--- | :--- |
| `missing-parameter` | 400 | `path` 未指定 |
| `invalid-path-format` | 400 | トラバーサル等の不正な形式 |
| `out-of-bounds-access` | 403 | `DATA_ROOT` 範囲外へのアクセス |
| `file-not-found` | 404 | 指定ファイルが存在しない |
